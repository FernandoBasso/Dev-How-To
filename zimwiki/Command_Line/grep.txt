====== grep ======
Created Friday 13 March 2020

Search for “eslint” ignoring certain files and directories:

'''
$ grep \
  --color \
  --recursive \
  --exclude=package*.json \
  --exclude-dir=coverage \
  --exclude-dir=node_modules \
  eslint
'''
