===== NVM Node Version Manager =====

Install latest LTS version:

'''
$ nvm install --lts
'''

Use the latest installed LTS:

'''
$ nvm use node --lts
'''

To make the latest LTS the default (one possible way shown here):

'''
$ nvm use node --lts
$ nvm alias default $(node --version)
'''
