==== Nonsensical Type Signature ====
This type signature is nonsense:

'''
fnC :: (Ord a, Ord b) => a -> b -> Bool
fnC x y = x > y
'''

Either ‘a’ or ‘b’ is ‘Ord’. They can't both be ‘Ord’. Fix:

'''
fnC :: Ord a => a -> a -> Bool
'''
